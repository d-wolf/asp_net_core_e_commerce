## Common CLI
### Create Multi Project Solution Core CLI
1. create a solution `dotnet new sln -o <SOLUTION_NAME>`
2. go into folder `cd <SOLUTION_NAME>`
3. create proj `dotnet new mvc -o ECommerce.<SOLUTION_PROJECT_NAME>`
